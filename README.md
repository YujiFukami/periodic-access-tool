# 定期アクセス支援ツール

クラウドソーシングサイト（Lancers・CrowdWorks など）へ、指定したURLに一定間隔で自動アクセスするWindows常駐デスクトップアプリです。

> 📖 **紹介記事**: [クラウドソーシングで「活動中」を自動で維持する方法](https://www.softex-celware.com/post/crowdsourcing-auto-login-tool)

## 主な機能

- **複数URL管理** — 管理名・URL・実行間隔・曜日・時間帯をURL単位で設定
- **定期自動実行** — 設定間隔でGoogle Chromeを起動してURLを表示
- **自動クローズ** — 指定秒数後に本アプリが開いたウィンドウのみを閉じる（他タブは影響なし）
- **タスクトレイ常駐** — メイン画面を閉じてもバックグラウンドで動作継続
- **実行ログ** — 日時・成功/失敗・エラー内容を記録。CSV出力対応
- **Windows自動起動** — 設定画面からPC起動時の自動起動を ON/OFF

## 動作環境

- Windows 10 / 11
- .NET 8.0 Runtime（または SDK）
- Google Chrome インストール済み

## インストール・起動手順

### 初回（ビルド）

```
build.bat をダブルクリック
```

`ビルドに成功しました。` と表示されれば完了。

### 通常起動

```
起動.bat をダブルクリック
```

または

```
bin\Release\net8.0-windows\定期アクセス支援ツール.exe をダブルクリック
```

起動するとタスクトレイ（画面右下）にアイコンが表示されます。

## 使い方

| 操作 | 方法 |
|------|------|
| メイン画面を開く | トレイアイコンをダブルクリック |
| URLを追加 | メイン画面「追加」ボタン |
| URLを編集 | 一覧で選択 → 「編集」ボタン |
| 今すぐ実行 | 一覧で選択 → 「今すぐ実行」ボタン |
| ログを確認 | メイン画面「ログ」ボタン |
| 設定変更 | メイン画面「設定」ボタン |
| アプリを終了 | トレイアイコン右クリック → 「終了」 |

## デフォルト登録URL

初回起動時に以下の2件が自動登録されます（30分間隔）。

| 管理名 | URL |
|--------|-----|
| Lancers マイページ | https://www.lancers.jp/mypage |
| CrowdWorks ダッシュボード | https://crowdworks.jp/dashboard |

## 設定項目

- **Windows起動時に自動起動** — PC起動と同時にアプリを常駐起動
- **通知設定** — 実行失敗時のバルーン通知 ON/OFF
- **ログ保存期間** — 古いログの自動削除日数
- **Chrome パス** — chrome.exe の場所（自動検出できない場合に指定）
- **最大同時実行数** — 同時に実行するURL数の上限
- **リトライ設定** — 失敗時の再試行回数・間隔

## データ保存場所

設定・ログは以下に保存されます。

```
%APPDATA%\定期アクセス支援ツール\
  settings.json   — アプリ設定
  entries.json    — URL登録データ
  logs.csv        — 実行ログ
```

## 注意事項

- 本ツールは「定期的にページを表示する」補助ツールです。ログイン状態の維持を保証するものではありません
- 各サービスの利用規約を確認の上、過剰なアクセスにならない間隔で使用してください
- ID/パスワードの自動入力・CAPTCHA突破などの機能は含まれていません

## 技術構成

- **言語**: C# (.NET 8)
- **UI フレームワーク**: Windows Forms
- **データ保存**: JSON（設定）、CSV（ログ）
- **Chrome制御**: Win32 API（ウィンドウハンドルで追跡・クローズ）
